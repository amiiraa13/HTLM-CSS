Explications de l'HTML/CSS pour ta daronne

C'est quoi ?

HTML, ou Hypertext Markup Language, est le langage de programmation utilisé pour créer et structurer des pages web. C'est un langage de balisage, ce qui signifie qu'il utilise des balises pour indiquer au navigateur web comment afficher le contenu. Les balises HTML entourent le contenu et définissent sa structure, comme les titres, les”’ paragraphes, les liens, les images, etc. En résumé, HTML est la fondation sur laquelle les pages web sont construites, permettant de créer une structure et de présenter du contenu sur Internet. C'est quoi ?

Le CSS, ou Cascading Style Sheets, est un langage utilisé pour styliser et mettre en forme les pages web créées en HTML. Contrairement à HTML qui se concentre sur la structure et le contenu, le CSS contrôle l'apparence visuelle des éléments HTML. Avec le CSS, tu peux définir des couleurs, des polices de caractères, des marges, des positions et d'autres propriétés visuelles pour chaque élément HTML sur une page web.
